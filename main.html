Quantifiers in Discrete Mathematics

Introduction

Quantifiers are symbols used in logic to express the quantity of specimens that satisfy a given condition. The two main types are universal and existential quantifiers.

Universal Quantifier

The universal quantifier (denoted by ∀) asserts that a predicate holds for all elements in a domain. It can be expressed in various forms:

  • For all x, p(x)
  • For every x, p(x)
  • Given any x, p(x)
∀x p(x)

Example:

The statement "For any real number x, we always have x² ≥ 0" can be expressed as:

∀x ∈ ℝ (x² ≥ 0)

Existential Quantifier

The existential quantifier (denoted by ∃) asserts that there is at least one element in a domain for which the predicate holds. It can be expressed as:

  • There exists an x such that p(x)
  • For some x, p(x)
∃x p(x)

Example:

The proposition "There exists a prime number x such that x + 2 is also prime" can be expressed as:

∃x (x is prime ∧ (x + 2) is prime)
